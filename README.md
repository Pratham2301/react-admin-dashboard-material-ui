# React Admin Dashboard | [Live Link][hostedLink]

## Technology used
1. React - Vite
2. Material UI 
3. Formik
4. Yup
5. Nivo chart
6. Full-calendar
7. React-Pro-Sidebar


```
yarn create vite

yarn dev
```


### This application consists of 
* Light & Dark Mode
* 4 different Charts
* 3 different Data Table Pages
* FAQ Page
* Form Page  
* Calendar Integration
